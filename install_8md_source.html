6 \section installation Installation
7 1. Install the dependencies (see \ref dependencies)
8 2. git submodule update --init
9 3. ./build.sh
10 4. ./setup-hugetlbfs.sh
11 5. Run MoonGen from the build directory
12 
13 Note: You can also use the script bind-interfaces.sh to bind all currently unused NICs (no routing table entry in the system) to DPDK/MoonGen. build.sh calls this script automatically. Use deps/dpdk/tools/dpdk_nic_bind.py to unbind NICs from the DPDK driver.
